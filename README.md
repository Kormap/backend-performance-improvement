# backend-performance-improvement

**backend-performance-improvement** 는 성능을 개선하는 방법과 부하 관련 테스트 내용을 제공합니다.

<br><br>

## 🔧 **기술 스택**

- **프레임워크:** Spring Boot 3.3.2
- **ORM:** JPA / QueryDSL
- **언어:** Java 17
- **데이터베이스:** PostgreSQL
- **API명세:** Swagger

<br><br>

## 🛠️ **주요 기능**
- **부하테스트:** ngrinder
- **모니터링:** 스카우트 모니터링

- **캐싱 조회 성능 개선:** 어플리케이션 레벨 캐싱(Ehcache) 사용
